# CT3xx WireViz Schema

Shared schema definitions for WireViz keys and UI type hints.

## Contents

- Known connector keys
- Known cable keys
- Property type hints (bool/array/number/string)
- Reference commit hash for WireViz syntax

- `Ct3xxWireVizParser`
