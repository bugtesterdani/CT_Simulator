CT3xxSimulator.Modules
======================

Gemeinsame Laufzeit fuer modul- und verdrahtungsnahe Simulatorbausteine.

Enthaelt aktuell:

- WireViz-Aufloesung und Pfadverfolgung
- Fault-Injection
- Runtime-Zielauflosung fuer Signale und Verbindungen

Dieses Projekt ist die vorgesehene Heimat fuer:

- SM2 / AM2 / AM4 / SC2 / SC3 / UIF Modulruntime
- testerseitige Hardware- und Kartenmodelle
- modulnahe Validierung und Pfadsemantik
