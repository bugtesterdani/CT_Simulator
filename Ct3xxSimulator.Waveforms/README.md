CT3xxSimulator.Waveforms
========================

Gemeinsame Waveform- und Recorder-Bausteine fuer den Simulator.

Enthaelt aktuell:

- `AppliedWaveform`
- `WaveformPoint`
- `ArbitraryWaveformLoader`

Dieses Projekt ist die vorgesehene Heimat fuer:

- AM2 / SA1 Waveform-Laden
- Recorder-/Stimulusmodelle
- gemeinsame Metrik- und Triggerauswertung
