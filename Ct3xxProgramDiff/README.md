# Ct3xxProgramDiff

Semantischer Vergleich von `.ctxprg`-Dateien auf Basis des internen Parser-Modells.

## Inhalt

Der Report listet:
- **Added**: neue Schritte/Tabellen in der neuen Datei
- **Removed**: Schritte/Tabellen aus der Referenz, die fehlen
- **Changed**: gleiche Knoten mit geaenderten Metadaten (Name, File, LogFlags, Parameter usw.)
